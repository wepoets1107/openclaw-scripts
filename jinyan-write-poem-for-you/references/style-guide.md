## 二、经典五律结构（按晋言原诗拆解）

### 【暮访山寺之一】
```
首联（起/叙事）：暮访灵光寺，松香槛外寻。
颔联（承/写景）：山风潜倦鸟，日影渡禅音。
颈联（转/感怀）：凝睇思忧喜，登临忘古今。
尾联（合/余韵）：白云归海去，为我洗尘心。
注：清明后，夫妻二人趁暮色登八大处...
```

### 【相思】
```
首联（起/点题）：杯酒遣相思，当吟侧帽词。
颔联（承/写景对仗）：襟前风逐月，纸上字成痴。
颈联（转/用典）：画扇悲公子，狂生慕紫芝。
尾联（合/回环）：缁尘京国客，杯酒遣相思。  ← 首尾重复，回环结构
注：写纳兰性德的相思...
```

### 【夏至夜与胖胖吃瓜有感】
```
首联（起/幽默）：溽暑赖何消，西瓜分外娇。
颔联（承/细节）：青青琢碧玉，默默饮寒刀。
颈联（转/调侃）：瓜血今朝洒，瓜皮明日抛。
尾联（合/荒诞升华）：年年瓜死处，奇志动云霄。
注：用新韵。
```

## 三、经典七律结构

### 【高铁返京车上作三首】（口语化代表作）
```
高铁隆隆困正浓，咽中生火腿生疼。
白云二朵飘窗外，一朵清纯一朵萌。
注：当代诗词流派中，有口语化诗词的主张。仿效口语诗，用网络语入诗，游戏之作，也别有一番趣味。用新韵。
```

### 【夜宿福州城有感】
```
首联（起）：独立危楼鸟绝飞，层云万里锁秋归。
颔联（承/对仗）：消磨一水西湖镜，摇曳千丝榕树帏。
颈联（转/对仗）：灯火连城人欲醉，江船别岸意难违。
尾联（合）：杯中琥珀浇肠热，裁剪相思正入微。
注：出差福州，夜宿有感。
```

### 【夏日候车时有感】
```
首联（起/问句）：人间俗事待如何，浮利虚名恁样多。
颔联（承/对仗）：懒散杂工得偷憩，殷勤行客似驱鹅。
颈联（转/对仗）：近来颠倒因罹病，远去湖山欲采荷。
尾联（合/俏皮）：交浅言深皆寂寞，不如闲唱一支歌。
```

### 【不惑之年辞职创业感怀兼寄诸师友】
```
首联（起叙事）：自出黉门入检门，它山攻玉已十春。
颔联（承感怀）：人生逆旅蓬蒿客，知己飘零旦暮身。
颈联（转自省）：满纸文章事权贵，一襟怀抱减天真。
尾联（合达观）：挣开尘网飞羁鸟，从此丘山共白云。
注：离职手续正式办完，从此天地一沙鸥，不惧风波恶。快哉！作诗记之。
```

## 四、常用词牌结构

### 浣溪沙（双调42字，上片三平韵，下片两平韵）
```典型结构：
上片：三句，第一句起，二三句承（写景或叙事）
下片：前两句对仗（工对），末句收束
```
**模板**（参照《浣溪沙·乌镇》）：
```
[七言起句]。[七言承句]。
[七言收束]。
[七言对仗上句]，[七言对仗下句]。
[七言收束/余韵]。
```

### 鹧鸪天（双调55字，上片四句三平韵，下片五句三平韵）
```典型结构：
上片：前两句写景起，三四句对仗（可宽）
下片：三字对两句，再接七言对仗，末句收束
```
**模板**（参照《鹧鸪天·南太行行吟》）：
```
[七言起]，[七言承]。
[七言对仗上]，[七言对仗下]。
[三言]，[三言]，[七言]。
[七言对仗上]，[七言对仗下]。
```

### 定风波（双调62字）
上片五句三平韵两仄韵，下片六句四仄韵两平韵——变化复杂，按谱填词。

## 五、注文写法模板

```
注：[时间/节气]，[地点/出行]，[人物关系/同行者]。[事件缘起]。[特殊说明/典故/出律解释]。[情感收尾]。
```

示例：
```
注：辛丑年大寒节气，刘建勋老师作诗，步韵唱和。后附原玉。
注：出差贵阳，报道贵州检察机关的智慧检务工作，深感"爽爽的贵阳"气候凉爽宜人。下午与友人绕观山湖长走，作诗记之。
注：夫妻二人异地生活，聚少离多。过年时，二人同去妻子家乡长沙，因聚而忧别，心事重重，故成此诗记之。
注：用新韵。
注：孤雁入群格。
```

## 六、步韵/次韵/唱和写法

### 步韵唱和结构
```
【次韵XXX原玉】
晋言

[依原诗韵脚依次押韵的诗作正文]

注：[交代唱和背景、附原玉]
```

示例 - 《林益涵&晋言：春日和诗合辑》15首：
两人各写15首，每首林益涵先发，晋言次韵。韵脚完全相同，内容各写各的生活和心境。这是晋言最标志性的创作方式。

### 步韵时的注意事项
1. **严格押原韵**：每句最后一个字的韵脚与原作完全一致
2. **次韵标注**：标题写"次韵XXX"或"步韵XXX"
3. **附原玉**：在注中附上原诗全文

## 七、禁用/慎用

- ❌ 不使用"老干部体"（空洞套话、政治口号）
- ❌ 不堆砌典故（用典须自然，生僻典加注）
- ❌ 不写无病呻吟（每首诗必须有真实事件/情感触发）
- ❌ 不要"陌上""天涯""红尘"三件套扎堆
- ⚠️ 俏皮话每首最多一处，不可过量
- ⚠️ 新韵必须标注，不可默认用新韵

## 八、诗作命名规范

```
【题目】← 五言/七言
【词牌名·题目】← 词
```

题目要求：
- 简洁直白，点明事件地点
- 多用"有感""感怀""杂感""偶题""口占""行吟"
- 含地名/人名：如《别福州返京寄诸友人》《贺孙兄伯光寿临耳顺》
- 节气+事件：如《近清明诸事加身有寄》《大寒步韵》
- 小辑用引号标注：如《"白云二朵飘窗外"诗小辑》
- 组诗标注数目：感秋四首、夜咏无题四首
